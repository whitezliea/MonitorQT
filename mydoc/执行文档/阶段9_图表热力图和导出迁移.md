# 阶段 9：图表、热力图和导出迁移

执行日期：2026-06-25

  - 支持实时缓存点和历史样本点两种数据源。
  - 显示预警/报警阈值线。
  - 显示非 Good 质量点标记。
  - 显示尖峰点标记和图例。

  - 新增 Realtime/History 数据源切换。
  - 新增趋势点 CSV 导出。
  - 保持 Tag、窗口、质量、尖峰诊断刷新逻辑一致。
- 扩展 `HeatmapWidget`：
  - 保持 16 x 16 网格自绘。
  - 支持选中单元格描边。
  - 支持异常点描边。
  - 新增自动/固定显示量程。
  - 新增工业热力色标图例。
- 扩展 `MeasurementMapPageWidget`：
  - 新增 Auto Range / Fixed Range 控件。
  - 新增热点信息显示。
  - 新增矩阵单元格 CSV 导出，包含值、显示归一化值、异常类型、严重级别、Tooltip 和量程模式。
- 新增 `presentation/export/CsvExportWriter.*`：
  - 统一写出 UTF-8 BOM。
  - 统一 CSV 字段双引号转义。
  - History、Trend、Matrix 导出共用同一写出规则。

## 行为对齐

- 趋势图渲染只消费 `UiSnapshot` 中的趋势/历史显示数据，不直接读取设备、模拟器或 SQLite。
- 热力图固定量程只影响显示归一化和颜色，不重新计算矩阵统计、异常点或业务质量状态。
- CSV 导出保持 Excel 友好的 UTF-8 BOM，字段始终按 CSV 规则双引号包裹并转义内部双引号。
- 矩阵异常点、热点文本和热力颜色均来自 Application DTO 或与 Application 色标一致的显示映射。
